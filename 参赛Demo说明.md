# GrowthMate 参赛 Demo 说明

## 一句话定位

GrowthMate 是面向普通 KOC 的 AI 增长 Agent，帮助用户从模糊人设出发，完成账号诊断、7 天增长任务规划、首条内容生成和发布后复盘，并通过 LLM Judge 评测平台持续迭代 Agent 质量。

## Demo 主线

1. 用户输入自己的 KOC 画像、赛道、平台、素材和当前卡点。
2. Agent 先做诊断，再给出账号定位、7 天增长任务和每条任务的观察信号。
3. 用户选择第一条任务，生成当前平台可发布的内容稿。
4. 用户发布后输入真实数据、评论样本和自己实际改过的发布稿。
5. Agent 基于真实发布稿做复盘，给出下一条内容任务。
6. 评测后台用固定评测集和 LLM Judge 评估输出质量，并把问题反向写回 Prompt。

## 3 分钟演示脚本

### 0:00-0:30 痛点和定位

普通 KOC 最大的问题不是不会写一句文案，而是不知道自己该做什么定位、第一周发什么、发完以后怎么判断要不要继续。GrowthMate 不是一个单点文案生成器，而是一个内容增长 Agent。

### 0:30-1:35 主应用工作流

演示用户填写画像：平台、赛道、人设、目标用户、已有素材、当前卡点。然后展示 Agent 输出：

- 诊断当前阻塞点
- 给出账号定位和打法
- 生成 7 天增长任务
- 每条任务绑定素材、观察信号和评论区预埋

再点进第一条任务，展示平台化内容生成。强调：GrowthMate 采用分步工作流，不一次性生成 7 天所有正文，而是先规划路径，再对用户当前选择的任务生成完整内容。

### 1:35-2:10 复盘闭环

用户不会原样发布 AI 文案，所以复盘时允许用户输入自己真实改过的发布稿、数据和评论样本。Agent 会区分用户真实发布稿和 Agent 建议稿，基于证据做归因，并生成下一条内容任务。

### 2:10-2:50 评测平台

展示 Eval Console：

- 评测集管理：5 个冷启动 case，覆盖不同平台、赛道和用户阶段
- 评测运行：LLM Judge 给出分数、理由、证据、问题和 Prompt 迭代建议
- 快照重评：同一批 Agent 输出可以复用新的 Judge Prompt 重评，不需要重新生成

重点讲一个真实迭代例子：美妆敏感肌 case 经过评测发现 note 正文缺失和复盘证据边界问题，修复后单 case 从 B 提升到 A。

### 2:50-3:00 总结

GrowthMate 的创新点是把 AI 从“生成内容”推进到“诊断、规划、生成、复盘、评测迭代”的闭环。它不仅能帮 KOC 产出内容，也能通过评测平台持续发现 Agent 的质量问题。

## 推荐展示的评测故事

### 评测集

冷启动阶段使用 5 个固定 case，覆盖：

- 小红书职场新人起号
- 抖音上班族减脂
- 视频号本地探店
- 公众号学习长文
- 小红书敏感肌护肤

选择逻辑：覆盖不同平台形态、账号阶段、素材类型、商业化敏感度和内容边界。

### 评测迭代闭环

1. LLM Judge 发现问题：早期 Judge 把“没有生成 Day2-Day7 完整正文”误判为缺陷。
2. 修正 Judge Prompt：明确 GrowthMate 是分步工作流，只要求 selectedTopic 有完整内容。
3. 复用已有输出重评：通过 `--judge-report` 对同一批 Agent 输出重新评审。
4. 反向改 Agent Prompt：加入关键记忆点、平台指标校验、真实发布稿和 Agent 建议稿边界。
5. 单 case 验证：`beauty_sensitive_xhs` 修复后达到 `85 / A`。

## 简历表达

GrowthMate - KOC 内容增长 AI Agent

- 设计并实现面向普通 KOC 的内容增长 Agent，覆盖账号诊断、7 天任务规划、内容生成和发布后复盘。
- 构建固定评测集与 LLM-as-Judge 评测后台，支持对 Agent 输出进行分维度评分、证据解释和 Prompt 迭代建议生成。
- 设计可复用评测快照机制，支持在不重新调用生成链路的情况下重评历史输出，用于区分 Agent 输出变化和评测标准变化。
- 通过评测结果反向迭代 Prompt 和规范化逻辑，修复平台指标混用、关键元素缺失、真实发布稿/AI 建议稿边界不清等问题。

## 后续可讲的 Roadmap

- 页面内异步评测任务队列，支持一键运行、失败重试和进度展示。
- 引入人工标注队列，用人工复核校准 LLM Judge。
- 收集真实用户回流数据，比较用户反馈和模型评审的一致性。
- 建立 Prompt 版本管理，追踪每次迭代对不同 case 的影响。
